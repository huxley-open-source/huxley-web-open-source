Para realizar uploads de casos de teste em lote, o arquivo deve seguir o seguinte padrão:

  • Todos os arquivos de um caso de teste deve seguir o mesmo prefixo;
  • A entrada deve possuir a extensão .in;
  • A saída deve possuir a extensão .out;
  • A dica deve possuir a extensão .tip;
  • Os casos de teste devem estar compactados em um só arquivo;

Atenção: a soma do tamanho do arquivo de entrada com o de saída deve ter no máximo 8mb

Exemplo:

Temos um problema com dois casos de teste. Os arquivos do primeiro caso estão organizados da seguinte maneira:

  • 1.in
  • 1.out
  • 1.tip

Já o segundo:

  • 2.in
  • 2.out

A estrutura do segundo caso de teste é um pouco diferente do primeiro, pois não possui dica. Os arquivos mandatórios para um caso de teste são a entrada e a saída.

Logo após os arquivos devem ser compactados em um só arquivo que pode ter qualquer nome.
